ng-phpdebugbar
==============

Make angular ajax requests work with PHP Debug Bar


````
bower install ng-phpdebugbar
````

Require the 'ng-phpdebugbar' module and you are all set

````
angular.module('myApp', ['ng-phpdebugbar'])
````
